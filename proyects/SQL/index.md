# Análisis de datos con SQL

Desarrollo de análisis de datos utilizando SQL como capa principal para la exploración, modelado y generación de métricas. Los proyectos demuestran el uso de consultas estructuradas para transformar datos crudos en datasets analíticos, listos para su consumo en herramientas de visualización como Excel o Power BI.

## Objetivo
Demostrar la capacidad para utilizar SQL como herramienta central de análisis, definiendo métricas consistentes, agregaciones, filtros y rankings que permitan responder preguntas de negocio de forma clara, reproducible y escalable.

## Habilidades demostradas
- Consultas SQL (SELECT, WHERE, ORDER BY, GROUP BY)
- Funciones de agregación (SUM, AVG, COUNT)
- Cálculo de métricas derivadas
- Análisis temporal
- Filtrado y segmentación de datos
- Uso de subconsultas y vistas
- Documentación y versionado de consultas
- Integración SQL → Excel / Power BI

## Metodología de trabajo
1. **Ingesta de datos**. Carga de información desde archivos planos, CRM o fuentes externas hacia una base de datos relacional.
2. **Preparación**. Validación de estructura, normalización de campos y control de calidad de los datos.
3. **Modelado**. Definición de tablas y campos clave para análisis.
4. **Análisis**. Construcción de consultas SQL para agregaciones, análisis temporal, rankings y métricas.
5. **Entrega**. Exportación de resultados para su visualización y análisis en herramientas BI.

## Tecnologías
- SQL
- SQLite (DB Browser for SQLite)
- Excel / Power BI (consumo de resultados)
