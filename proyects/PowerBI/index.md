# Reportes y dashboards en Power BI

Desarrollo de dashboards interactivos orientados al análisis de datos operativos y comerciales, utilizando Power BI para la visualización, modelado y análisis de información. Los proyectos están diseñados para facilitar la toma de decisiones mediante KPIs claros, segmentación avanzada y modelos de datos estructurados.

## Objetivo
Demostrar la capacidad para construir soluciones de Business Intelligence que integren datos de distintas fuentes, transformen la información en indicadores accionables y presenten resultados de forma clara, interactiva y orientada al negocio.

## Habilidades demostradas
- Modelado de datos
- Power Query (ETL)
- DAX básico e intermedio
- Construcción de KPIs
- Dashboards interactivos
- Segmentación y filtros
- Análisis descriptivo y comparativo
- Visualización orientada a negocio

## Metodología de trabajo
1. **Ingesta de datos**. Conexión a archivos, bases de datos o fuentes operativas; carga y consolidación.
2. **Transformación (Power Query)**. Limpieza, normalización, tipado de datos y creación de tablas auxiliares.
3. **Modelado**. Definición de relaciones, jerarquías y modelo en estrella.
